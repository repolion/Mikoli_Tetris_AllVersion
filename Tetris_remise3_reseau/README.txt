Projet Tetris 
==========		

Remise 3
------------


Cette remise comprend:


- Une documentation complète du code au format Html ainsi qu'au format pdf
- Les headers et les classes sources non  compilées 
- Les ressources:  images, musique, sons,...
- Un rapport complet sur l'état actuel du projet (version pdf & docx)

Le projet est fonctionnel et à été testé sur différents environnements.


Les touches utilisées pour jouer sont:
	- "flèches gauche", "droite", "bas" pour le déplacement
	- "w" pour la rotation à gauche
	- "x" pour la rotation à droite
	- "barre d'espace" pour éffectuer un drop
	- "p" pour mettre le jeu en pause


Programmé par Mike Sarton et Olivier Cordier
